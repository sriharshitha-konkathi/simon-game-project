# My Simon Game Project 👋

**🚀 Live Demo: [View Project on GitHub Pages](https://sriharshitha-konkathi.github.io/simon-game-project/)**

Hi there! This is a Simon Game project I built to practice and showcase my frontend development skills.

## 🛠️ What's inside?
Here's a quick look at the tech stack I chose to build this project:

- **HTML** for structure.
- **CSS** for responsive styling.
- **JavaScript (Vanilla)** for handling the game logic and DOM manipulation.

## 🚀 How to run it locally
Want to try it out on your own machine? It's super easy!

1. Clone the repository:
   ```bash
   git clone https://github.com/sriharshitha-konkathi/simon-game-project.git
   ```
2. Open the `index.html` file in your favorite web browser to start playing!
